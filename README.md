# Password Generator

## Overview
This project is a command-line password generator developed using Python as part of the CodSoft Python Programming Internship. It creates random passwords of a user-defined length by combining alphabetic characters and numeric digits.

## Features
- Generates passwords of custom length
- Uses uppercase and lowercase letters
- Includes numeric digits
- Simple command-line interface

## Technologies Used
- Python

## How to Run
1. Run the `password-generator.py` file.
2. Enter the required password length.
3. The program will generate and display a random password.
